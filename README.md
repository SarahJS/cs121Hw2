# README

Instagram-like website created for cs121 Homework 2, following this tutorial:
https://www.youtube.com/watch?v=MpFO4Zr0EPE

### Ruby version

2.4.0

### Rails version

5.0.0

### Functionality Included:

* Login
* Register
* Create Posts
* Styling added using bootstrap grid classes (http://getbootstrap.com/css/#grid)
* Grey background added, plus some padding between the posts.
* Changed the size of photos added so that at least 3 fit in each row.
